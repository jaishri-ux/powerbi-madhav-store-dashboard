# Madhav Store Sales Dashboard (Power BI)

This project presents an interactive Power BI dashboard created to analyze sales, profit, and order performance of Madhav Store.

## Key Features
- KPIs: Total Sales, Profit, Quantity, Average Order Value
- Analysis by State, Category, Sub-Category, Customer, and Payment Mode
- Quarter-wise and region-wise filtering using slicers

## Tools Used
- Power BI
- Power Query
- DAX
- Excel (dataset)
